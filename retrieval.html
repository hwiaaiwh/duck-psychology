Source Amnesia

  • Inability to recall origin of memory but retain the main information
  • E.g. You remember who the first president was, but forgot who told you

Retrieval Cues

Priming

  • Activation of memory through association
  • E.g. An old song bringing you back in time

State Dependent Memory

  • What's learned in one state is easily remembered when in the same state
  • E.g. Learning something drunk will be better recalled when drunk

Context Dependent Memory

  • Memory of something that is remembered better in its original context
  • E.g. Taking a test in the same classroom you learned the material

Serial Position Effect

  • The tendency to recall the first and last term in a series
  • E.g. Remebering the first and last items on a grocery list, whilst forgetting the middle items
